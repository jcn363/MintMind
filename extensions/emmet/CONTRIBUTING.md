## How to build and run from source?

Read the basics about extension authoring from [Extending MintMind](https://code.visualstudio.com/docs/extensions/overview)

- Read [Build and Run MintMind from Source](https://github.com/microsoft/vscode/wiki/How-to-Contribute#build-and-run-from-source) to get a local dev set up running for MintMind
- Open the `extensions/emmet` folder in the vscode repo in MintMind
- Press F5 to start debugging

## Running tests

Tests for Emmet extension are run as integration tests as part of MintMind.

- Read [Build and Run MintMind from Source](https://github.com/microsoft/vscode/wiki/How-to-Contribute#build-and-run-from-source) to get a local dev set up running for MintMind
- Run `./scripts/test-integration.sh` to run all the integrations tests that include the Emmet tests.
